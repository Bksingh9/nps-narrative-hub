ℹ️ About These Exports

These CSV files contain detailed payment transaction data from the Kafka payment status topic. The data includes comprehensive information about each transaction including: order IDs, payment amounts, customer details, transaction status, payment methods, billing/delivery addresses, cart information, and Gringotts response details. You can open these files in Excel, Google Sheets, or any data analysis tool for further investigation.

Key Fields: order_id, payment_status, amount, customer_email, transaction_status, merchant_order_id, payment_method, gringotts_message, and many more...
